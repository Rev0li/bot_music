# Icônes de l'Extension

Pour créer les icônes, vous pouvez:

   - Générer les tailles 16x16, 48x48, 128x128

2. **Utiliser Photoshop/GIMP:**
   - Créer une image avec une note de musique
   - Exporter en PNG aux tailles:
     - icon16.png (16x16)
     - icon48.png (48x48)
     - icon128.png (128x128)

3. **Utiliser des icônes gratuites:**
   - https://www.flaticon.com/ (chercher "music note")
   - Télécharger et redimensionner

## Couleurs Suggérées

Pour correspondre au thème de l'extension:
- Couleur principale: `#667eea` (bleu-violet)
- Couleur secondaire: `#764ba2` (violet)
- Fond: Transparent ou blanc

## Fichiers Requis

- `icon16.png` - Icône dans la barre d'outils
- `icon48.png` - Icône dans la page des extensions
- `icon128.png` - Icône dans le Chrome Web Store
